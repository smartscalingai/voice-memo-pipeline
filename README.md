# Voice Memo Transcription Pipeline

Automated voice memo transcription pipeline powered by Claude Code routines.

**Flow:** Record voice memo → Share to Google Drive → Claude routine transcribes → Saved to Notion

## Components
- **Google Drive:** Voice Memos folder (source)
- **Claude Code Routine:** Scheduled daily transcription
- **Notion Database:** Voice Memos (searchable transcript store)
